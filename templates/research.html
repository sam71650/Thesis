How FGSM Works
Step 01
Compute the gradient of the loss J with respect to the input pixels — not the model weights. This tells us how each pixel affects the loss.
∇ₓ J(θ, x, y)
Step 02
Take the sign of each gradient value — producing a matrix of +1 or −1 per pixel, giving the direction of steepest loss increase.
sign( ∇ₓ J)
Step 03
Scale by ε and add to the original image — injecting invisible noise that maximally fools the model while staying imperceptible to humans.
x + ε · sign(∇ₓ J)
FGSM (Fast Gradient Sign Method) is a one-step white-box attack introduced by Goodfellow et al. (2014). It crafts an adversarial image by nudging each pixel in the direction that increases the model's loss — making the face unrecognisable to the classifier while remaining visually identical to a human eye. The perturbation is controlled by ε: larger values fool the model more aggressively but become more visible.
FGSM Formula
x_adv = x + ε · sign( ∇ₓ J( θ, x, y))
x — original input image x_adv — adversarial image ε — perturbation magnitude ∇ₓ J — gradient of loss w.r.t. input θ — model parameters y — true label
